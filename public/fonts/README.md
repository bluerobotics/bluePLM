# Tengwar Elvish Font

To enable the Elvish language Easter egg, download a Tengwar font and place it here.

## Recommended Font

**Tengwar Annatar** by Johan Winge - the most popular and complete Tengwar font.

1. Download from: https://www.dafont.com/tengwar-annatar.font
2. Extract the ZIP file
3. Copy `tngan.ttf` (or `TengwarAnnatar.ttf`) to this folder
4. Rename it to `TengwarAnnatar.ttf` if needed

The font will automatically be used when you select "Sindarin (Elvish)" from the language menu.

## Alternative Fonts

- **Tengwar Quenya**: https://www.dafont.com/tengwar-quenya.font
- **Tengwar Eldanaro**: https://www.dafont.com/tengwar-eldanaro.font

## How It Works

When Sindarin is selected, the entire UI switches to the Tengwar font, making the app completely unreadable (but beautifully Elvish). This is a hidden Easter egg for Tolkien fans!
